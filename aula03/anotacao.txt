npm init -Y -> inicia o projeto

npm i "nome do pacote"  instala o pacote

npm i "nome do pacote" -E instala o pacote porem o mesmo nunca será atualizado

npm i "nome do pacote" --save-dev  instala o pacote como dependencia de desenvolvimento

npm i "nome do pacote" --save-prod  instala o pacote como dependencia do projeto  

npm update   atualiza o pacote 

npm i  "nome do pacote"@versao do pacote instala a versão especifica 

npm uninstall "nome dos pacotes
